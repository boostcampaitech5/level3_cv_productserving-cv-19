## Usage

Mac OS version<br/>
Install : Node.js, JAVA 11 SDK, X Code, CocoaPods, watchman, Android Studio, Typescript

```bash
$ git clone https://github.com/boostcampaitech5/level3_cv_productserving-cv-19.git
$ npm install
$ npm run android
```
